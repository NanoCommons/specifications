Intended audience

This document is aimed at everyone working in the EU NanoSafety Cluster.

This document provides mappings for identifiers originally used by the NanoMILE project (FP7, Grant number 310451). The content of this specification is collected within the NanoSolveIT project. The source of this page can be found at https://github.com/nanocommons/specifications/nanomile/.

Introduction

Because NanoMILE material data has been reused in other project, including NanoSolveIT [[Afantitis2020]], and materials tested in other projects, other identifiers are available, like for the European Registry of Materials [[VanRijn2022]]. Currently, no research output has been identified to provide information about the materials. When found, these will be indexed using Wikidata and then Wikidata identifiers will be added.
