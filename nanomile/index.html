Intended audience

This document is aimed at everyone working in the EU NanoSafety Cluster.

The sources can be found at https://github.com/nanocommons/specifications.

Introduction

This document provides mappings for identifiers originally used by the NanoMILE project (FP7, Grant number 310451). Because NanoMILE material data has been reused in other project and materials tested in other projects, other identifiers are available, like for the European Registry of Materials [[VanRijn2022]].
