Intended audience

This document is aimed at people who want to convert a data set into ISA-TAB-Nano.

ISA-TAB-Nano has been identified as a potential common exchange format for nanosafety data. Originally developed in the USA, it is now being adopted by various European NanoSafety Cluster projects too. This note describes the various steps involved on starting with some data set and ends with a conversion into ISA-TAB-Nano.

Introduction

Converting one data set to another is more difficult when more semantics are being introduced. A guide to help you and explain what the things are one should think about is helpfull. This has previously been done for other semantic formats, such as RDF in general [[Marshall2012]].
